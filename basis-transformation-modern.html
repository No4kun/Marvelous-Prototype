基底の定義と条件

ベクトル空間 V の基底 B = {v₁, v₂, ..., vₙ} は以下を満たします：

$$\begin{align} &\text{1. 線形独立性: } c_1v_1 + c_2v_2 + \cdots + c_nv_n = 0 \Rightarrow c_1 = c_2 = \cdots = c_n = 0 \\ &\text{2. 生成性: } \text{span}(\mathcal{B}) = V \end{align}$$
